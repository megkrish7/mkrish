DiaTrack - Advanced Diabetic Care

Problem Statement

Managing diabetes effectively requires constant monitoring of health metrics, which can be particularly challenging for the elderly who prefer simplicity in technology usage. Many elderly diabetic patients seek an all-in-one application that not only helps track their daily health data but also simplifies the process of finding local pharmacies with insulin in stock. The need for a straightforward user interface that accommodates limited tech-savviness without compromising on functionality is critical in empowering these patients to manage their condition efficiently.

Solution Developed

DiaTrack is designed to serve as a comprehensive health management tool specifically tailored for diabetics, especially catering to the needs of the elderly. Developed with a focus on simplicity and accessibility, the application aims to tackle the following problems:

  • Comprehensive Health Tracking: Users can easily input and monitor their blood glucose levels, dietary intake, and medication schedules.
  • Machine Learning Predictions: Utilizes historical data to predict future health trends and glucose levels, aiding in proactive health management.
  • Local Pharmacy Search: Integrates a KNN clustering algorithm to locate nearby pharmacies that have insulin in stock, displayed via an interactive map.

This solution not only simplifies daily health tracking but also enhances the autonomy of elderly patients in managing diabetes, potentially reducing emergency situations and improving overall health outcomes.

Technical Challenges and Solutions

Data Management and Security

Challenge: Securely storing and managing sensitive health data within an easy-to-use application for elderly users.

Solution: We implemented SQLite to handle data storage with robust encryption protocols to ensure data security and privacy. The backend, built with Node.js, provides a secure and scalable server environment.

User Interface Simplicity and Accessibility

Challenge: Creating an intuitive and accessible interface that elderly users find easy to navigate.

Solution: The frontend, developed using React.js, offers a clean, straightforward layout with large buttons and text, ensuring high usability and accessibility.

Real-Time Pharmacy Stock Updates

Challenge: Providing up-to-date information on insulin availability at local pharmacies.

Solution: Integrating the KNN clustering algorithm with real-time data feeds from pharmacies allows the app to display current insulin stock levels effectively.
